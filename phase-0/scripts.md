# Phase 0 — Generated Scripts

2 variants per product. Same prompt structure for all: Hook (problem) → Transformation → CTA.
Products 4 and 5 have zero reviews — scripts are weaker as a result. Flagged below.
Use the same avatar for both variants of the same product — format is the variable being tested, not demographics.

---

## Phase 0 Findings

**Tool:** Seedance 2 via TopView.ai video agent. Reference photos were pasted directly and prompts were built from the scripts below. Seedance generated 1–3 clips per prompt and stitched them together.

**Video quality:** Seedance 2 is the only model tested with quality high enough for this to work. Other models are not viable.

**Audio:** ElevenLabs audio outperformed Seedance 2's native audio. Recommended pipeline going forward: generate ElevenLabs audio first → generate Seedance video without audio → combine in post.

**What worked:** B-roll of the user or the product (or both) with a voiceover or text overlay on top. Any format that does not require lip sync. Follow-me, first-person POV, and aesthetic b-roll all fall into this category.

**What didn't work:**
- Talking head style (creator speaking directly to camera) — quality was acceptable but not good enough. Avoid for now.
- Green screen style (creator talking with green screen effect, images or video behind them) — did not work.

**Text overlays:** Seedance 2 cannot reliably render text inside video. Any prompt that calls for on-screen text requires a separate text overlay step applied after video generation.
